# 🚗 IoT-Based Vehicle Health Monitoring System

An embedded IoT system for monitoring important vehicle parameters using ESP32 and multiple sensors.

## Features

- Engine Temperature Monitoring
- Battery Voltage Monitoring
- Obstacle Detection
- Real-time Alerts
- MQTT Cloud Communication
- Proteus Simulation

## Hardware

- ESP32
- DHT11 / LM35
- Ultrasonic Sensor
- Buzzer
- OLED Display

## Software

- Embedded C
- Arduino IDE
- MQTT
- Proteus
